 The Gematria values themselves currently have not been useful in solving the Liber Primus.
 Calculations are done modulo 29, so if a value goes over 28 or under 0, it wraps around to the other side.
 The Atbash cipher swaps the first character in the alphabet with the last, the second character with the penultimate, and so on.
 The Vigenere cipher uses a specific key, and shifts each value according to the key. In this case it can be reversed by subtracting the key values from the ciphertext.
 e.g.
 ᚢᛠᛝᛋᛇᚠᚳᚱᛇᚢᚷᛈᛠᛠᚠᚹᛉᛏᚳᛚᛠ
- DIVINITYDIVINITYDIVIN
= WELCOMEWELCOMEPILGRIM
 In order to make them more difficult to solve, multiple ciphers skip over values of F (ᚠ, 0) in the plaintext. This means that some of the F's visible in the ciphertext were left unchanged, but not all of them. It would be difficult to find which indices these were without using trial and error. The positions of the F's that you should skip are given. Treat the skipped F's as though they do not affect the cipher in any way, i.e. do not go to the next prime number/vigenere key position.

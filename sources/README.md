# Sources & Provenance

Every research file under `research/` carries its own `## Sources` section with URLs.

## Primary source archives
- **Uncovering Cicada Wiki** (community-maintained, the canonical solving archive) — the most complete record of every puzzle step.
- **Cicada 3301 PGP-signed messages** — the only statements treated as authentically "from 3301" are those signed with the 3301 PGP key (fingerprint published in the puzzles). Anything unsigned is suspect.

## Reliability tiers (how this repo weighs sources)
1. **Primary artifacts** — the actual images, audio, runic pages, and PGP-signed text. Highest trust.
2. **Community solving wikis / archives** — high trust for puzzle *mechanics*, lower for *identity* claims.
3. **Reputable journalism** — Rolling Stone, The Washington Post, The Atlantic, Mental Floss, Vox. Good for context; weak on identity (no one has confirmed it).
4. **Forums / Discord / YouTube / blogs** — useful leads, treat as rumor until corroborated.

## Misinformation warning
Cicada 3301 identity claims are heavily polluted by self-promoters, hoaxers, and litigious disputes. The `identity/` folder separates *documented fact* from *contested claims*. No source has ever credibly confirmed who runs 3301.
